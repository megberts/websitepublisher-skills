# WebsitePublisher Skills

Agent skills for building websites with WebsitePublisher.ai via MCP.

| Skill | Description |
|-------|-------------|
| `websitepublisher-api` | Core API skill — page templates, SAPI forms, image editor, go-live checklist |
| `websitepublisher-design` | Design guidelines and component patterns |
| `websitepublisher-dev` | Full development workflow for advanced site builds |

## Usage

These skills are used by AI coding agents (Claude Code, Cursor, Windsurf) to build websites on the WebsitePublisher platform. Each folder contains a `SKILL.md` with instructions.

## MCP Server

Connect via: `https://mcp.websitepublisher.ai`
